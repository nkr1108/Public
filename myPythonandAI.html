Combining Dataframes

  • Using concat method
    • pd.concat([df1,df2],axis=0) # returns new dataframe concat by rows
    • pd.concat([df1,df2],axis=1) # returns new dataframe concat by columns
  • Using merge method
    • pd.merge(df1,df2,how='outer',on='CommonColumn') # returns new dataframe, works like outer join on given column
    • pd.merge(df1,df2,how='inner',on='CommonColumn') # returns new dataframe, works like inner join on given column
    • pd.merge(df1,df2,how='right',on='CommonColumn') # returns new dataframe, works like right join on given column
    • pd.merge(df1,df2,how='left',on='CommonColumn') # returns new dataframe, works like left join on given column
  • Using Join Method - works similar to merge method
    • df1.join(df2,how='outer') # returns new dataframe, works like outer join on given column
    • df1.join(df2,how='inner') # returns new dataframe, works like inner join on given column
    • df1.join(df2,how='right') # returns new dataframe, works like right join on given column
    • df1.join(df2,how='left') # returns new dataframe, works like left join on given column

Data Wrangling

  • Check Duplicate Values
    • df.duplicated().sum() # returns total number of duplicate rows
    • df.drop_duplicates(inplace=True) # returns clean data after dropping duplicates
    • df.reset_index(drop=True,inplace=True) # resets the index
  • Missing Values, ? or blank or NA or 0 or NaN
    • df.isnull().sum() # returns total number of missing values by column
    • replace '?' with NaN => df = df.replace('?',np.NaN)
    • df['col1'] = data['col1'].replace(['missing','inf'],np.NaN)
    • replace NaN with mean => df['col1'].replace(np.NaN,df['col1'].mean(),inplace=True) # mean is impacted by presence of outliers so median is preferred for such cases. For categorical data, model is preferred
    • drop row with NaN => df.dropna(subset=['col'],axis=0,inplace=True)
  • Data format Issue
    • df['price'] = df['price'].astype('int') # could be float
    • df['date'] = pd.to_datetime(df['date'])
    • df['col1'].apply(type).value_counts() # returns counts of value types
  • Data normalization (centering/scaling) - changes values from zeros to one
    • Scaling => new value = old/max
    • Min-Max => new value = (old-min)/(max-min)
    • Z-Score => new value = (old-average(meu))/(Standard Deviation(sigma)) (-3 to 3)
  • Data binning - convert numerical into categorical variables, like age - into 1-5, 5-10 or so on
    • bins = np.linspace(min(df['age']),max(df['age']),4)
    • group_names = ['low','medium','high']
    • df['age-binned']=pd.cut(df['age'],bins,labels=group_names,include_lowest=True)
  • Categorical values into numerical - One-hot encoding - gas and diesel, add 2 new columns/features and assign 1 and 0
    • pd.get_dummies(df['fuel'])

Quantitative / Numerical Variable

  • Univariate Analysis - Analysing one Variable
    • displot() - Visualize the distribution of variable (also called histogram)
    • boxplot() or violinplot() - To check specifically for outliers
  • Bivariate Analysis – Relationship between 2 variables
    • joinplot() or pairplot()
    • lmplot() – Scatter plot with a best fit line
  • Multivariate Analysis – Relationship between more than 2 variables
    • corr() – Correlation matrix followed by
    • heatmap() – Visualize the correlation matrix
    • pairplot() – Combination of Scatter plots and individual histogram plots for all numerical variables in the dataset. Also, can assign a categorical variable using hue as an add on)

Qualitative / Categorical Variable

  • Univariate Analysis – Analysing one variable
    • countplot() – Visualize the distributions of categorical variable
  • Quantitative vs Qualitative Variables - Analyse how a quantitative variable varies across categorical variable(s)
    • boxplot() or violinplot() – To check specifically for outliers
    • stripplot() or swarmplot() – Scatter plot across a categorical variable (also helps in checking for outliers)
    • barplot() – Can also create a clustered bar chart (assign a categorical variable to hue) or a stacked bar chart (2 bar plots with different colors)
    • pointplot()
    • lineplot() – Best when looking at trends (Time-related variable along the x-axis)
    • catplot() or factorplot() – Analysing a quantitative variable across 2 categorical variables with one variable having a high number of categories
